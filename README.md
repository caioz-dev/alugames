# 🎲 AluGames

Aplicação web para simular um sistema de aluguel e devolução de jogos 
de tabuleiro. O usuário pode alugar jogos disponíveis e devolvê-los 
quando quiser, com atualização visual do status em tempo real.

## 🚀 Funcionalidades

- Listagem de jogos disponíveis para aluguel
- Alugar e devolver jogos com um clique
- Atualização visual do status do jogo (disponível/alugado)
- Interface interativa e responsiva

## 🛠️ Tecnologias

- HTML5
- CSS3
- JavaScript (manipulação do DOM)
